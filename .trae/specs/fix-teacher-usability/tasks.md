# Tasks

## 阶段一：核心功能增强

- [ ] Task 1: 添加题目批量导入功能
  - 创建题目导入 API (`POST /api/exams/<id>/questions/import`)
  - 在 teacher.html 添加"批量导入"按钮和模态框
  - 支持 Excel/CSV 格式解析
  - 添加题目模板下载功能
  - **验证**: 上传 Excel 文件成功导入多道题目

- [ ] Task 2: 提供答题卡模板下载
  - 创建答题卡模板 HTML 文件
  - 在扫描页面添加"答题卡模板"下载按钮
  - 模板包含学号填写区、选择题区域、填空题区域
  - **验证**: 点击下载获得可打印的答题卡

- [ ] Task 3: 成绩逐题调整功能
  - 修改审核详情，题目得分可点击编辑
  - 添加单题分数修改 API
  - 分数修改后同步更新总分
  - **验证**: 点击题目分数可修改，总分自动更新

- [ ] Task 4: 成绩批量调整功能
  - 添加多选功能（复选框）
  - 添加批量加分/减分功能
  - 添加批量直接设分功能
  - **验证**: 选中多个学生后可批量调整

- [ ] Task 5: 成绩调整备注功能
  - 添加成绩调整记录表结构
  - 记录调整人、时间、原因、原分数、新分数
  - 在成绩详情中显示调整历史
  - **验证**: 调整成绩后可查看调整记录
