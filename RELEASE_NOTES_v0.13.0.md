# v0.13.0 - 用户体验优化版

## 📝 更新说明

本次更新重点优化用户体验，新增简化版教师端、AI智能识别、批量操作功能，让老师的日常教学工作更加便捷高效。

## 🆕 新增功能

### 1. 简化版教师端

为老师提供更加简洁高效的操作界面，适合日常教学使用。

**访问地址**: `/simplified`

**主要功能**：
- 📝 **快速录分** - 直接输入或粘贴成绩，最快1分钟完成
- 📷 **答题卡批改** - AI自动识别，支持批量处理
- 📊 **成绩管理** - 查看历史成绩、导出分析报告

**使用场景**：
- 随堂测验成绩录入
- 周测、月考成绩管理
- 日常教学数据记录

### 2. AI智能答题卡识别

支持多种识别模式，灵活适应不同的教学场景。

**三种识别模式**：

| 模式 | 上传材料 | 识别方式 | 适用场景 |
|------|---------|---------|---------|
| 精确批改 | 原试卷 + 标准答案 + 答题卡 | AI智能分析 | 正式考试 |
| 自动批改 | 标准答案 + 答题卡 | 自动比对 | 有标准答案 |
| 手写识别 | 仅答题卡 | 识别手写分数 | 已批改答题卡 |

**技术特点**：
- 批量处理多张答题卡
- 自动识别学号和姓名
- 支持选择题、填空题识别
- 识别置信度提示

### 3. 学生成绩查询系统

为学生提供便捷的成绩查询入口。

**访问地址**: `/student`

**功能特点**：
- 📝 输入学号快速查询
- 📈 成绩趋势图表展示
- 🧠 知识点掌握度分析
- 📋 单次考试详细分析

**展示内容**：
- 所有历史考试成绩
- 成绩变化趋势图
- 各知识点掌握情况
- 班级排名信息

### 4. 批量操作功能

大幅提升教师工作效率。

**批量确认成绩**
- 一键确认多条成绩
- 支持按条件筛选
- 批量标记优秀/及格

**批量调整分数**
- 固定分数调整
- 按比例调整
- 加分/减分调整
- 实时预览效果

### 5. 成绩导出功能

支持多种格式导出，方便数据管理。

**导出格式**：
- 📄 CSV格式 - 兼容性好，适合导入其他系统
- 📊 Excel格式 - 格式美观，包含统计信息

**导出内容**：
- 单次考试成绩
- 班级成绩汇总
- 详细统计报告

## 🔧 技术改进

### API接口增强
- `POST /api/exams/<id>/scores` - 批量添加成绩
- `POST /api/exams/<id>/batch-confirm` - 批量确认
- `POST /api/exams/<id>/batch-adjust` - 批量调整
- `GET /api/exams/<id>/export` - 成绩导出
- `GET /api/student/<学号>/scores` - 学生成绩查询
- `POST /api/scan/score` - 答题卡识别

## 🚀 快速开始

### 1. 访问简化教师端
```
http://localhost:5000/simplified
```

### 2. 快速录分流程
```
选择班级 → 输入考试名称 → 输入/粘贴分数 → 保存发布
```

### 3. 答题卡批改流程
```
1. 创建考试（名称、班级、满分）
2. 上传材料（可选）
3. 批量上传答题卡
4. 等待AI识别
5. 审核确认 → 发布
```

### 4. 学生查询成绩
```
http://localhost:5000/student
输入学号 → 查询成绩
```
