# Math_Correcter - 数学智能批改系统

一个旨在让数学学习更平等和更精准的项目，为教师和学生提供完整的数字化教学解决方案。

## 📋 功能特性

### 教师端功能

#### 🆕 简化教师端 (`/simplified`)
- 📝 **快速录分** - 直接输入或粘贴成绩，适合日常测验，最快1分钟完成
- 📷 **答题卡批改** - AI自动识别手写分数或自动批改，支持批量上传
- 📊 **成绩管理** - 查看历史成绩、导出分析报告

#### 完整教师端 (`/teacher`)
- 🏫 **班级管理** - 创建班级、添加学生、管理学生名单
- 📝 **考试管理** - 创建考试、录入题目、设置正确答案
- 🔍 **智能扫描** - 自动识别答题卡、考号、答题结果
- ✅ **批量操作** - 批量确认成绩、批量调整分数、标记优秀/及格
- 📊 **学情分析** - 班级统计、错题率分析、知识点掌握热力图

### 学生端功能 (`/student`)
- 📝 **成绩查询** - 输入学号快速查询个人所有成绩
- 📈 **趋势分析** - 查看成绩变化趋势图
- 🧠 **知识点掌握** - 查看各知识点掌握情况分析
- 📋 **详情查看** - 查看单次考试详细题目分析

### 核心功能
- 📊 **成绩导出** - 支持CSV/Excel格式导出
- 🔐 **安全验证** - 文件上传验证、API密钥环境变量配置
- 🌐 **跨平台** - 支持Linux、macOS、Windows

## 🚀 快速开始

### 1. 环境准备

```bash
# 克隆项目
git clone https://github.com/yourusername/Math_Correcter.git
cd Math_Correcter

# 安装依赖
pip install -r requirements.txt
```

### 2. 启动系统

```bash
python server.py
```

### 3. 访问地址

- **🆕 简化教师端**: http://localhost:5000/simplified （推荐，日常使用）
- **教师端**: http://localhost:5000/teacher
- **学生端**: http://localhost:5000/student
- **试卷库**: http://localhost:5000/library
- **管理后台**: http://localhost:5000/admin

## 📝 使用流程

### 🆕 简化教师端使用流程（推荐）

#### 快速录分（日常测验）
```
1. 选择班级 → 输入考试名称 → 输入/粘贴分数 → 保存发布
```
适合：随堂测验、周测、月考

#### 答题卡批改（正式考试）
```
1. 创建考试（名称、班级、满分）
2. 上传材料（可选：原试卷、标准答案）
3. 批量上传答题卡
4. AI自动识别分数
5. 审核确认 → 一键发布
```
支持三种识别模式：
- 📄 原试卷 + 标准答案 → 精确批改
- 📋 只有标准答案 → 自动批改
- 📷 只有答题卡 → 识别手写分数

### 完整教师端使用流程

1. **班级管理** - 创建班级，添加学生（学号必填）
2. **创建考试** - 填写考试信息，录入题目和正确答案
3. **设置就绪** - 确认题目无误，设置考试为就绪状态
4. **扫描答题卡** - 批量上传学生答题卡，系统自动识别
5. **审核成绩** - 查看扫描结果，调整分数，批量确认
6. **确认入库** - 确认成绩无误，提交到系统
7. **学情分析** - 查看班级整体分析和知识点掌握情况

## 🔧 API接口

### 班级管理
- `GET /api/classes` - 获取所有班级
- `POST /api/classes` - 创建班级
- `GET /api/classes/<id>` - 获取班级详情
- `POST /api/classes/<id>/students` - 添加学生
- `DELETE /api/classes/<id>` - 删除班级

### 考试管理
- `GET /api/exams` - 获取所有考试
- `POST /api/exams` - 创建考试
- `GET /api/exams/<id>` - 获取考试详情
- `POST /api/exams/<id>/scores` - 添加成绩（新增）
- `POST /api/exams/<id>/confirm` - 确认成绩入库
- `POST /api/exams/<id>/adjust` - 调整单条成绩
- `POST /api/exams/<id>/batch-confirm` - 批量确认成绩（新增）
- `POST /api/exams/<id>/batch-adjust` - 批量调整分数（新增）
- `GET /api/exams/<id>/export` - 导出成绩CSV/Excel（新增）
- `GET /api/exams/<id>/analysis` - 获取考试分析

### 学生端（新增）
- `GET /api/student/<学号>/scores` - 获取学生所有成绩
- `GET /api/student/<学号>/exam/<考试ID>` - 获取学生考试详情

### 答题卡识别（新增）
- `POST /api/scan/score` - AI识别答题卡分数

## 📊 核心功能说明

### 智能答题卡识别
- 自动识别学生考号
- 识别每道题的答题结果
- 自动对比正确答案
- 计算分数和正确率
- 知识点掌握度统计

### 学情分析
- 班级平均分、及格率、优秀率
- 每道题的错题率（帮助发现教学重点）
- 知识点掌握热力图（红色=需强化，绿色=掌握良好）
- 学生进步趋势分析

## 🛠️ 技术栈

- **后端**: Flask (Python)
- **前端**: HTML5 + Tailwind CSS + JavaScript
- **数据存储**: JSON文件（轻量级，无需数据库）
- **AI引擎**: 支持LLM集成（processor.py）

## ⚠️ 注意事项

1. **学号必须准确** - 系统通过学号匹配学生
2. **答题卡格式** - 建议使用标准答题卡，学生信息区域清晰
3. **正确答案录入** - 创建考试时必须录入正确答案才能自动批改
4. **审核环节** - 扫描结果必须人工审核后才能入库
5. **安全配置** - 建议设置环境变量 `DASHSCOPE_API_KEY` 存储API密钥
6. **跨平台兼容** - 系统支持Linux、macOS、Windows，所有路径自动处理
