# 教师阅卷系统 - 使用说明

## 📋 系统概述

这是一个完整的教师阅卷系统，支持：
- 班级与学生管理
- 考试创建与题目录入
- 答题卡扫描与自动识别
- 成绩审核与录入
- 学情分析与报告生成

## 🚀 快速开始

### 1. 启动系统

```bash
cd /workspace
python server.py
```

### 2. 访问地址

- **教师端**: http://localhost:5000/teacher
- **学生端**: http://localhost:5000/library

> 注意：需要管理员登录

## 📝 完整工作流程

### 步骤1: 班级管理

1. 进入"班级管理"模块
2. 创建班级（填写班级名称、年级）
3. 打开班级详情
4. 添加学生（必须填写学号，姓名可选）

### 步骤2: 创建考试

1. 进入"考试管理"模块
2. 点击"创建考试"
3. 填写考试信息：
   - 考试名称（例如：期中数学考试）
   - 选择班级
   - 设置总分
4. 添加题目（可添加多道）：
   - 题型：choice（选择题）、true_false（判断题）、fill_blank（填空题）、subjective（主观题）
   - 分值：每题分数
   - 正确答案：例如 A, B, C, D 或 √, ×
   - 知识点：可添加多个，用逗号分隔
5. 点击"创建并设为就绪"

### 步骤3: 扫描答题卡

1. 进入"答题卡扫描"模块
2. 选择已就绪的考试
3. 拖拽上传所有学生的答题卡图片
4. 点击"开始扫描"
5. 系统自动识别并显示结果

### 步骤4: 成绩审核

1. 进入"成绩审核"模块
2. 选择"待审核"状态的考试
3. 查看扫描结果：
   - 已匹配：已识别到学号的学生
   - 未匹配：无法识别学号的答题卡
4. 可对单条成绩进行调整
5. 确认无误后，点击"确认所有成绩"

### 步骤5: 学情分析

1. 进入"学情分析"模块
2. 选择"已完成"的考试
3. 查看详细分析：
   - 班级整体统计（平均分、及格率等）
   - 各题错题率
   - 知识点掌握情况

## 🎯 核心功能

### 答题卡识别

系统会自动识别：
- 学生考号（关键识别字段）
- 每道题的答题结果
- 自动对比正确答案
- 计算分数和正确率

### 人工审核机制

所有扫描结果都需要教师人工审核后才能入库，确保准确性。

### 学情分析

- 班级整体成绩统计
- 每道题的错题率（帮助发现教学重点）
- 知识点掌握热力图

## 📂 数据存储

所有数据存储在 `data/` 目录下：

```
data/
├── classes/
│   └── metadata.json      # 班级和学生数据
├── exams/
│   └── metadata.json       # 考试数据
├── reports/              # 生成的报告
├── test_library/          # 试卷库
│   ├── images/           # 上传的答题卡图片
│   └── scans/           # 扫描的图片
└── ...
```

## 🔧 API接口

系统提供完整的RESTful API：

### 班级管理
- `GET /api/classes` - 获取所有班级
- `POST /api/classes` - 创建班级
- `POST /api/classes/<id>/students` - 添加学生

### 考试管理
- `GET /api/exams` - 获取所有考试
- `POST /api/exams` - 创建考试
- `GET /api/exams/<id>` - 获取考试详情
- `POST /api/exams/<id>/questions` - 添加题目
- `POST /api/exams/<id>/ready` - 设置就绪
- `POST /api/exams/<id>/scan` - 扫描答题卡
- `POST /api/exams/<id>/confirm` - 确认成绩
- `GET /api/exams/<id>/analysis` - 获取分析

## ⚠️ 注意事项

1. **学号必须准确**：系统通过学号匹配学生，请确保学号准确无误
2. **答题卡格式**：建议使用标准答题卡，学生信息区域清晰
3. **正确答案**：创建考试时必须录入正确答案才能进行自动批改
4. **审核环节**：扫描结果必须人工审核后才能入库

## 💡 使用技巧

1. **批量添加学生**：可以在创建班级后，批量导入学生名单
2. **题目模板**：可以保存常用考试模板，快速创建考试
3. **成绩对比**：可以查看学生历史成绩，分析进步情况
4. **知识点分析**：根据错题率调整教学重点

## 🆘 故障排除

### 无法访问页面
- 检查Flask服务是否正常运行
- 检查端口5000是否被占用

### 无法识别学号
- 检查答题卡中学生信息区域是否清晰
- 确认学号格式与录入时一致

### 数据丢失
- 检查 `data/` 目录权限
- 定期备份重要数据
